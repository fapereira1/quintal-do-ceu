Como Se Tornar Voluntário

Siga estes 4 passos simples para começar sua jornada como voluntário:

1

Inscreva-se

Preencha nosso formulário de cadastro com suas informações e áreas de interesse.

  • ✓ Dados pessoais básicos
  • ✓ Áreas de interesse
  • ✓ Disponibilidade de horários
  • ✓ Experiências anteriores
2

Entrevista

Participaremos de uma conversa para conhecer melhor seu perfil e expectativas.

  • ✓ Conversa online ou presencial
  • ✓ Alinhamento de expectativas
  • ✓ Definição do projeto ideal
  • ✓ Cronograma de atividades
3

Capacitação

Oferecemos treinamento específico para que você esteja preparado para contribuir.

  • ✓ Treinamento específico do projeto
  • ✓ Orientações de segurança
  • ✓ Conhecimento sobre a comunidade
  • ✓ Certificação de voluntário
4

Ação

Comece a fazer a diferença nos projetos que mais combinam com seu perfil!

  • ✓ Início das atividades práticas
  • ✓ Acompanhamento contínuo
  • ✓ Suporte da equipe
  • ✓ Celebração dos resultados
